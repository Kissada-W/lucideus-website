# Technology Stack

## Core
- **Language:** TypeScript
- **Runtime:** React 19
- **Framework:** Next.js 16 (App Router)
- **Package Manager:** Bun

## Styling & UI
- **Styling:** Tailwind CSS v4
- **Theming:** CSS Variables, `next-themes` (Dark/Light mode)
- **Component Library:** shadcn/ui (Radix UI primitives)
- **Icons:** Lucide React

## Animation
- **Library:** Framer Motion

## Quality & Standards
- **Linting:** ESLint
- **Formatting:** Prettier (assumed via project conventions)
